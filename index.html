Programação na WEB

Na Escola de Comercio de Lisboa, na disciplina de Tecnologia 1 no 3ºA TIS, iremos aprender Programação WEB 😅 e abordaremos os principios básicos associados ao desenvolvimento para a WEB, apenas do lado front-end, do lado visual. Támbem sabemos que existe um lado back-end, que é o lado do servidor, mas não iremos abordar esse lado.

Aprender-se-ão as seguintes tecnologias 🌐:

  • HTML
  • CSS
  • JavaScript

De forma a consolidar todos os conhecimentos, srão feitos alguns laboratorios.!!!
